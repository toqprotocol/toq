# Contributing to toq protocol

Thank you for your interest in contributing to toq protocol.

## Getting Started

1. Clone the repository
2. Make sure you have Rust installed via [rustup](https://rustup.rs/). The correct toolchain version will be installed automatically from `rust-toolchain.toml`.
3. Build: `cargo build`
4. Run tests: `cargo test`
5. Check formatting: `cargo fmt --check`
6. Run lints: `cargo clippy`

## Pull Requests

- Fork the repo and create your branch from `main`.
- Run `cargo fmt`, `cargo clippy`, and `cargo test` before submitting.
- Write clear commit messages.
- Keep PRs focused. One change per PR.

## Reporting Issues

Open an issue on GitHub. Include steps to reproduce, expected behavior, and actual behavior.
